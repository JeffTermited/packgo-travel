# 雄獅旅遊網站設計分析

## 首頁搜尋欄設計

### 標籤切換設計
- 兩個紅色標籤按鈕：「關鍵字搜尋」和「目的地搜尋」
- 標籤位於搜尋欄上方，讓使用者選擇不同的搜尋方式
- 選中的標籤為紅色背景，未選中為白色背景

### 搜尋欄位排列
- **橫向單行排列**：出發地、關鍵字、出發區間、搜尋按鈕全部在同一行
- 欄位之間緊密排列，沒有過多間距
- 搜尋按鈕為紅色圓形按鈕，帶有放大鏡圖示

### 勾選框位置
- 「只找成行」和「只找可報名」兩個勾選框
- 位於搜尋欄下方，與搜尋欄對齊
- 勾選框為白色背景，帶有綠色勾選標記

### 右側功能區
- 「熱門搜尋」和「首頁一鍵」按鈕位於搜尋欄右側
- 提供快速訪問常用功能

### 整體視覺風格
- 搜尋欄背景為半透明白色
- 輸入框為白色背景，帶有淺灰色邊框
- 標籤文字清晰，字體大小適中
- 整體設計簡潔專業

## 關鍵設計要點
1. 標籤切換功能讓使用者選擇搜尋方式
2. 橫向單行排列讓所有欄位一目了然
3. 勾選框位於搜尋欄下方，不佔用主要空間
4. 右側功能區提供額外便利性
5. 視覺風格統一，配色協調


## 分析總結

由於無法直接訪問雄獅旅遊的搜尋結果頁面，但從首頁搜尋欄和您提供的截圖，我們可以總結以下關鍵設計原則：

### 搜尋欄設計原則
1. **標籤切換功能**：提供「關鍵字搜尋」和「目的地搜尋」兩種模式
2. **橫向單行排列**：所有主要欄位（出發地、關鍵字、出發區間、搜尋按鈕）在同一行
3. **緊湊但不擁擠**：欄位之間間距適中，視覺整齊
4. **勾選框獨立行**：「只找成行」和「只找可報名」在搜尋欄下方獨立一行
5. **右側功能區**：「熱門搜尋」和「首頁一鍵」等快捷功能

### 搜尋結果頁面設計原則（根據截圖分析）
1. **搜尋欄固定在頂部**：方便使用者隨時調整搜尋條件
2. **左側篩選器**：使用可折疊的區塊設計，兩欄式 checkbox 佈局
3. **右側行程列表**：行程卡片排列整齊，資訊層次分明
4. **視覺層次清晰**：標題、價格、特色標籤等元素對比明顯

### 應用到我們網站的建議
1. 保持搜尋欄在搜尋結果頁面頂部固定
2. 優化篩選器的視覺層次和間距
3. 確保所有元素對齊一致
4. 加入標籤切換功能（如果需要多種搜尋方式）
5. 保持極度寬敞的排版，避免擁擠感
