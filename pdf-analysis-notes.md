## 實作方案

### PDF 解析流程
1. 用戶上傳 PDF 檔案
2. 將 PDF 轉換為圖片（每頁一張）
3. 使用 LLM Vision 分析每頁內容
4. 提取結構化資料（標題、價格、行程、圖片等）
5. 從 PDF 中提取圖片資源
6. 生成行程資料並儲存

### 技術實作
- 使用 pdf-lib 或 pdf2pic 將 PDF 轉為圖片
- 使用 pdf-image-extractor 提取 PDF 中的圖片
- 使用 LLM Vision API 分析頁面內容
- 將提取的圖片上傳到 S3
