# Expedia 日期選擇器設計研究

## 研究目標
參考 Expedia 的日期選擇器設計，為 PACK&GO 旅行社網站實作統一的日期選擇器元件。

## Expedia 日期選擇器特色

### 視覺設計
1. **彈出式日曆**：點擊日期輸入框後，彈出完整的月曆視圖
2. **雙月顯示**：同時顯示兩個月份，方便選擇跨月日期
3. **清晰的視覺層次**：
   - 當前日期有特殊標記
   - 選中的日期範圍有背景色高亮
   - 不可選日期（過去日期）顯示為灰色
4. **圓潤設計**：日期數字使用圓形背景，符合現代設計趨勢

### 互動設計
1. **一體化體驗**：
   - 「去程」和「回程」日期在同一個日曆中選擇
   - 選擇去程日期後，日曆自動保持開啟，等待選擇回程日期
   - 選擇完成後自動關閉日曆
2. **即時反饋**：
   - Hover 效果明顯
   - 選擇範圍即時顯示
3. **快捷選項**：提供「彈性日期」等快捷選項

### 功能特點
1. **日期範圍選擇**：支援選擇開始和結束日期
2. **最小停留天數**：可設定最小和最大停留天數限制
3. **響應式設計**：在手機和桌面版都有良好體驗

## 實作建議

### 技術選擇
推薦使用 `react-day-picker` 套件，因為：
- 輕量級且高度可客製化
- 支援日期範圍選擇
- 有良好的 TypeScript 支援
- 可完全控制樣式

### 設計規範（符合 PACK&GO 黑白風格）
1. **配色**：
   - 背景：白色
   - 邊框：灰色 (border-gray-300)
   - 選中日期：黑色背景，白色文字
   - Hover 效果：淺灰色背景
2. **圓角**：使用 rounded-xl 保持與網站整體風格一致
3. **陰影**：使用 shadow-xl 增加層次感

### 元件結構
```
DateRangePicker
├── Trigger (輸入框)
├── Popover (彈出層)
└── Calendar (日曆視圖)
    ├── Month Navigation
    ├── Week Days Header
    └── Days Grid
```
