# Sipincollection.com 每日行程頁面設計分析

參考頁面：https://sipincollection.com/tour/daily/26EI315TK1-T/detail?fr=cg451T0401C0101M01

## 一、整體設計特點

### 1. Hero 區域
- **全螢幕 Hero 圖片**：義大利托斯卡尼山城風景
- **右側資訊卡片**（米色背景）：
  - 標題：「義大利典藏冬情14日」
  - 副標題：「2晚威尼斯五星連泊」
  - 天數標籤：「14 Days」
  - 出發/返國日期
  - 價格：$239,900
  - 兩個按鈕：「立即報名」（白色）、「看程特色」（綠色邊框）

### 2. 航班資訊區塊
- 標題：「航班資訊 Flight information」
- 表格顯示：
  - 去程/回程
  - 搭乘班機（航空公司 + 航班號）
  - 出發地/抵達地
  - 日期/時間
  - 飛行時數
- **景點距離參考**：列出各城市間的距離

### 3. 每日行程區塊

#### 左側導航
- 固定在左側的天數選擇器（Day 1 - Day 14）
- 當前選中的天數高亮顯示（黃色背景）

#### 主要內容區域

**每日標題**：
- 大標題：「台北 ✈ 羅馬」
- 副標題：「啟程 Departure」
- 詩意文案：「乘著雪白的羽翼伴隨悸動期待的心情，啟程...」

**景點介紹**（重點！）：
- **每個景點一個區塊**
- **左右交錯佈局**：
  - Day 1: 左圖右文 → 右圖左文 → 左圖右文
  - 圖片佔 50% 寬度
  - 文字佔 50% 寬度
- **景點標題**：
  - 主標題：「龐貝古城 Pompei」
  - 副標題（詩意）：「火山灰封、時光凝結的歷史見證—龐貝古城，千年火山巨瞳下的羅馬繁華，訴說維蘇威火山下的永恆一瞬與滄桑。」
- **景點描述**：100-200 字的深度介紹（故事性文案）
- **標籤**：「入內參觀」（小標籤）

**美食資訊**：
- 圖示 + 文字：早餐、中餐、晚餐

**住宿資訊**：
- 圖示 + 文字：飯店名稱 + 星級

### 4. 配色主題
- **主色**：米色/淺棕色（#C9B8A8）
- **輔助色**：白色、深棕色
- **強調色**：綠色（按鈕）、黃色（選中狀態）

### 5. 字體設計
- **標題**：襯線字體（優雅）
- **內文**：無襯線字體（易讀）
- **英文副標題**：斜體

## 二、關鍵設計元素

### 1. 詩意文案
每個景點都有兩層標題：
1. **主標題**：景點名稱（中英文）
2. **詩意副標題**：例如「火山灰封、時光凝結的歷史見證—龐貝古城」

### 2. 深度介紹
每個景點的描述都是 100-200 字的故事性文案，例如：
> 「沒有任何地方能像龐貝一樣，將時光瞬間凝固。西元79年，維蘇威火山的驟然噴發，將這座繁華的羅馬城市封存於火山灰下...」

### 3. 圖片與文字交錯
- 不是單純的圖片輪播
- 每個景點都有專屬圖片
- 圖片與文字交錯排列（左圖右文 → 右圖左文）

### 4. 行程下載功能
- 「行程下載」按鈕（綠色邊框）
- 可下載完整行程 PDF

## 四、實作重點

1. **Web Scraper Agent**：從雄獅旅遊等網站抓取原始資料
2. **AttractionAgent**：生成景點深度介紹（100-200 字）
3. **HotelAgent**：生成飯店介紹
4. **MealAgent**：生成餐食資訊
5. **FlightAgent**：生成航班資訊
6. **PoeticAgent**：生成詩意副標題和文案
7. **Image Generation Agent**：為每個景點生成圖片
8. **Master Agent**：協調所有 Agent

## 五、前端實作重點

1. **Hero 區域**：全螢幕圖片 + 右側資訊卡片
2. **航班資訊表格**：清晰的表格設計
3. **左側天數導航**：固定在左側，可快速跳轉
4. **景點區塊**：左右交錯佈局（50% 圖片 + 50% 文字）
5. **詩意副標題**：每個景點都有
6. **美食/住宿資訊**：圖示 + 文字
7. **行程下載按鈕**：生成 PDF
