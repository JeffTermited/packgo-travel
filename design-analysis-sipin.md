# 璽品旅遊行程詳情頁面設計分析

參考網址：https://sipincollection.com/tour/feature/25JHPHNH6/detail

## 整體設計風格

### 1. 配色方案
- **主色調**：根據目的地特色動態調整
  - 北海道行程：藍色系（天空藍、湖水藍）
  - 配色溫和優雅，營造高端質感
- **輔助色**：灰色、白色、金色點綴
- **文字色**：深灰色、白色（覆蓋在圖片上時）

### 2. Hero 區域設計
- **全屏大圖背景**：展示目的地主要景觀
- **置中標題**：大字體行程名稱
- **副標題**：行程亮點摘要（用點號分隔）
- **浮動導航**：深色半透明導航條
  - 左側：Logo
  - 中間：行程名稱、特色介紹、出發日與價格
  - 右側：更多相關行程、行程諮詢、出團日期篩選

### 3. 內容區塊設計

#### 特色介紹區塊
- **大型直式文字排版**：
  - 左側：超大字體的關鍵詞（直式排列）
    - 例如：「雅奢旅宿」
  - 右側：橫向圖片 + 詩意文案
    - 例如：「覽秘境無邊風月」「品其國美饌名湯」「享星鑰洞迴匠心」
- **圖文交錯排版**：左右交替展示
- **留白充足**：營造高端質感

#### 行程亮點區塊
- **卡片式設計**：每個亮點一張卡片
- **圖片為主**：大圖展示景點或體驗
- **簡潔文字**：標題 + 簡短描述
- **統一圓角**：所有卡片和圖片都有圓角

### 4. 出發日期與價格區塊
- **表格式呈現**：
  - 出團日期
  - 行程名稱
  - 航空公司
  - 售價
  - 報名狀態（感謝額滿/立即預約）
  - 備註
  - 每日行程連結
- **清晰的視覺層次**：表頭深色背景，內容白色背景

### 5. 視覺元素特點
- **大量留白**：營造呼吸感
- **圖片質量高**：專業攝影作品
- **文字排版優雅**：使用直式排版增加藝術感
- **圓角設計**：所有卡片、按鈕、圖片都有圓角
- **半透明效果**：導航條、文字背景使用半透明

## 實作計劃

### 階段一：資料庫擴充
- [ ] 在 tours 表新增欄位：
  - `heroImage`：Hero 區域大圖
  - `colorTheme`：主題配色（JSON 格式，包含主色、輔助色）
  - `highlights`：行程亮點（JSON 陣列，每個亮點包含圖片、標題、描述）
  - `keyFeatures`：關鍵特色（JSON 陣列，用於大字直式排版）

### 階段二：圖片生成與上傳
- [ ] 整合 Manus 圖片生成 API
- [ ] 根據行程目的地和特色生成 Hero 圖片
- [ ] 生成行程亮點圖片
- [ ] 上傳圖片至 S3 並儲存 URL

### 階段三：配色系統
- [ ] 建立配色生成邏輯：
  - 根據目的地特色選擇主色調
  - 例如：日本 → 櫻花粉/富士藍、歐洲 → 古典金/優雅紫
- [ ] 實作動態配色系統（CSS 變數）

### 階段四：前端重構
- [ ] 重新設計 TourDetail 頁面：
  - Hero 區域（全屏大圖 + 置中標題）
  - 浮動導航條
  - 特色介紹區塊（大字直式排版 + 圖文交錯）
  - 行程亮點卡片
  - 出發日期與價格表格
- [ ] 實作響應式設計
- [ ] 加入動畫效果（滾動視差、淡入淡出）

### 階段五：AI 輔助內容生成
- [ ] 使用 LLM 生成詩意文案
- [ ] 根據行程特色生成關鍵詞（用於大字排版）
- [ ] 生成行程亮點描述

## 補充設計細節

### 6. 圖片展示卡片
- **三圖並排**：ONSEN（溫泉）、LOBBY（大廳）、STAY（住宿）
- **圓角卡片**：圖片上方圓角，下方有金色標籤
- **金色標籤**：右下角斜角標籤，英文大寫標題
- **統一尺寸**：所有卡片等寬等高

### 7. 直式文字排版
- **右側大字**：「遊」「特別安排」等關鍵詞
- **左側小字**：詩意描述或說明文字
- **留白充足**：文字周圍有大量留白
- **字體選擇**：使用優雅的中文字體（可能是明體或宋體）

### 8. 配色細節（北海道行程）
- **主色**：深藍色（#2C3E50 類似色）
- **輔助色**：天空藍（#4A90E2 類似色）
- **點綴色**：金色（#F39C12 類似色）
- **背景色**：淺灰色（#F5F5F5）、白色
- **文字色**：深灰色（#333333）

### 9. 互動元素
- **浮動導航**：滾動時保持在頂部
- **平滑滾動**：點擊導航項目時平滑滾動到對應區塊
- **Hover 效果**：按鈕和連結有 hover 效果
- **圖片 Lazy Loading**：圖片延遲載入優化性能

### 10. 響應式設計
- **桌面版**：三欄並排、大圖展示
- **平板版**：兩欄並排、調整間距
- **手機版**：單欄排列、堆疊顯示
