# 台灣行程類型知識庫

## 鳴日號火車行程

### 識別特徵
- 標題包含「鳴日號」
- 提及「南港站」「花蓮站」「台東站」等車站
- 提及「車廂體驗」「緩行」「景觀」
- 提及「觀光列車」「鐵道旅行」

### 關鍵約束
- **交通方式：必須是火車**（絕對不是飛機）⚠️
- 集合地點：通常為南港火車站
- 特色體驗：車廂導覽、車上餐盒、緩行觀景

### 常見飯店
- The GAYA Hotel 潮渡假酒店（台東）
- 花蓮潔西艾美渡假酒店
- 理想大地渡假飯店
- 娜路彎大酒店
- 台東桂田喜來登酒店

⚠️ **重要**：這些飯店名稱不可互相替換！必須使用原始資料中指定的飯店。

### 常見景點
- 普悠瑪部落（卑南族文化體驗）
- 成功海光走讀（金曲製作人導覽）
- 如豐琢玉工坊（台灣玉DIY）
- 鹿野高台（熱氣球）
- 三仙台（跨海步橋）
- 石城-大里（緩行遠眺龜山島）
- 東里站（原住民歌舞表演）

### 識別邏輯

```typescript
function isMingriTrainTour(title: string, content: string): boolean {
  const searchText = (title + ' ' + content).toLowerCase();
  
  return searchText.includes('鳴日號') ||
         searchText.includes('觀光列車') ||
         (searchText.includes('火車') && 
          (searchText.includes('台東') || searchText.includes('花蓮')));
}
```

## 一般火車行程

### 識別特徵
- 標題包含「火車」「鐵路」「高鐵」「台鐵」
- 不特指「鳴日號」
- 提及車站名稱

### 關鍵約束
- 交通方式：火車、高鐵、台鐵
- 不能改為飛機或巴士

### 常見路線
- 台北-花蓮（太魯閣號、普悠瑪號）
- 台北-台東（南迴線）
- 環島鐵路

## 郵輪行程

### 識別特徵
- 標題包含「郵輪」「遊輪」「cruise」
- 提及「登船」「岸上觀光」「海上」
- 提及郵輪名稱（如「歌詩達」「公主號」「皇家加勒比」）

### 關鍵約束
- **交通方式：必須是郵輪**（不是飛機）⚠️
- 住宿：通常在郵輪上（艙房）
- 餐食：通常包含在郵輪上

### 常見港口
- 基隆港
- 高雄港
- 花蓮港
- 那霸港（日本）
- 石垣島港（日本）

## 自駕行程

### 識別特徵
- 標題包含「自駕」「租車」「開車」
- 提及「GPS」「自由行」
- 提及租車公司

### 關鍵約束
- 交通方式：自駕、租車
- 通常沒有導遊
- 行程彈性較大

### 常見路線
- 花東縱谷
- 南迴公路
- 北海岸
- 阿里山公路

---

## 飛機行程

### 識別特徵
- 標題包含「飛機」「航班」「機票」
- 提及機場名稱
- 提及航空公司

### 關鍵約束
- 交通方式：飛機
- 需要機場接送
- 有航班時間限制

### 常見航線
- 台北松山-花蓮
- 台北松山-台東
- 高雄-花蓮
- 高雄-台東

## 通用活動關鍵字

以下活動通常是合理的，即使原始資料未明確提及：

### 行程開始/結束
- 抵達、返程、出發、集合
- 入住、退房
- 機場/車站接送

### 餐食相關
- 早餐、午餐、晚餐
- 下午茶、宵夜
- 自理

### 交通相關
- 專車接送
- 步行
- 搭乘（火車/巴士/飛機）

## 行程類型識別流程

```
1. 檢查標題是否包含「鳴日號」
   → 是：MINGRI_TRAIN（交通=火車）
   
2. 檢查標題是否包含「郵輪」「遊輪」
   → 是：CRUISE（交通=郵輪）
   
3. 檢查標題是否包含「自駕」「租車」
   → 是：SELF_DRIVE（交通=自駕）
   
4. 檢查標題是否包含「火車」「鐵路」「高鐵」
   → 是：TRAIN（交通=火車）
   
5. 檢查標題是否包含「飛機」「航班」
   → 是：FLIGHT（交通=飛機）
   
6. 以上都不是
   → GENERAL（交通=待確認）
```

## 常見錯誤案例

### 案例 1：鳴日號行程被改為飛機

**原始資料**：
```
標題：鳴日號台東縱谷雲海漫遊3日
交通：鳴日號觀光列車
```

**錯誤生成**：
```
交通方式：飛機
出發地：台東機場
```

**正確生成**：
```
交通方式：火車（鳴日號）
出發地：南港火車站
```

### 案例 2：飯店名稱被替換

**原始資料**：
```
Day 1 住宿：The GAYA Hotel 潮渡假酒店
Day 2 住宿：花蓮潔西艾美渡假酒店
```

**錯誤生成**：
```
Day 1 住宿：娜路彎大酒店
Day 2 住宿：理想大地
```

**正確生成**：
```
Day 1 住宿：The GAYA Hotel 潮渡假酒店
Day 2 住宿：花蓮潔西艾美渡假酒店
```

### 案例 3：景點被創造

**原始資料**：
```
景點：普悠瑪部落、成功海光走讀、如豐琢玉工坊
```

**錯誤生成**：
```
景點：鹿野高台、三仙台日出、七星潭踏浪
```

**正確生成**：
```
景點：普悠瑪部落、成功海光走讀、如豐琢玉工坊
```
