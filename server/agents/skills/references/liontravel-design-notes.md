**來源**: https://travel.liontravel.com/

## 一、配色方案

### 主色調
- **品牌紅色**: #E60012 (雄獅 Logo 紅)
- **深灰色**: #333333 (主要文字)
- **淺灰色**: #666666 (次要文字)
- **背景白色**: #FFFFFF
- **淺灰背景**: #F5F5F5 (區塊分隔)

### 輔助色彩
- **橙色標籤**: #FF6600 (促銷標籤)
- **藍色按鈕**: #0099FF (搜尋按鈕)
- **綠色狀態**: #00CC66 (成團可報名)
- **灰色狀態**: #999999 (暫時額滿)

### 色彩使用原則
- 大量留白,背景以白色為主
- 品牌紅色僅用於 Logo 和重要 CTA
- 標籤和狀態使用多種顏色區分(非黑白)
- 價格使用大號紅色數字突出

## 二、字體系統

### 字體家族
- **中文字體**: Microsoft JhengHei (微軟正黑體) / Noto Sans TC
- **英文字體**: Arial, Helvetica, sans-serif
- **數字字體**: Arial (價格數字)

### 字體大小層級
- **標題**: 18-24px (行程名稱)
- **正文**: 14-16px (描述文字)
- **小字**: 12px (標籤、日期)
- **價格**: 24-32px (粗體紅色)

### 字體使用原則
- 標題使用粗體(font-weight: 700)
- 正文使用常規體(font-weight: 400)
- 行距: 1.5-1.8 (確保可讀性)

## 三、排版結構

### 頁面佈局
- **Header**: 固定頂部導航 + 品牌 Logo
- **Search Panel**: 搜尋欄位 + 篩選條件
- **Content Area**: 行程卡片列表(左側篩選 + 右側列表)
- **Footer**: 聯絡資訊 + 公司資訊

### 行程卡片設計
- **卡片尺寸**: 寬度 100%, 高度自適應
- **卡片間距**: 16px 垂直間距
- **卡片內容**:
  - 左側: 行程編號 + 天數標籤
  - 中間: 行程圖片 (16:9 比例)
  - 右側: 行程資訊 + 價格 + 出發日期

### 間距系統
- **容器內距**: 16px / 24px / 32px
- **元素間距**: 8px / 12px / 16px
- **區塊間距**: 24px / 32px / 48px

## 四、圖片使用風格

### 圖片類型
- **行程主圖**: 16:9 橫向照片
- **目的地風景**: 自然風光、地標建築
- **活動照片**: 遊客體驗、特色活動
- **美食照片**: 當地特色料理

### 圖片品質要求
- **解析度**: 至少 1200x675px (16:9)
- **檔案格式**: JPEG / WebP
- **色彩飽和度**: 中高飽和度,色彩鮮豔
- **構圖**: 主體突出,背景簡潔

### 圖片風格特徵
- 明亮、鮮豔、吸引人
- 展現目的地特色和魅力
- 避免過度後製或濾鏡
- 真實感 + 美感平衡

### 按鈕樣式
- **主要按鈕**: 藍色背景 + 白色文字 + 圓角 4px

  - 經典: 橙色
  - 特選: 藍色

  - 限時優惠: 紅色

### 輸入框樣式
- **邊框**: 1px solid #DDDDDD
- **圓角**: 4px
- **高度**: 40px
- **內距**: 12px
- **焦點狀態**: 藍色邊框 + 陰影

## 六、內容風格

### 標題風格
- **直述式標題**: 清楚說明行程內容
- **關鍵字堆疊**: 用「.」或「|」分隔亮點
- **範例**: "泰國清邁旅遊｜渡假村2晚.金三角遊船.天空步道.白廟.友善象營"

### 描述文字
- **簡潔明瞭**: 重點突出,避免冗長
- **關鍵資訊**: 天數、出發地、價格、特色
- **標籤輔助**: 用標籤快速傳達行程特點

### 資訊層級
1. **行程名稱** (最重要)
2. **價格** (次重要)
3. **出發日期 + 狀態** (重要)
4. **標籤 + 特色** (輔助)

## 九、行程詳情頁設計分析

### 標題格式
**範例**: "旗艦北海道旅遊｜保住雙百選溫泉.小樽貴賓館.秘境青池.小樽遊船.騎馬體驗.日本和牛.帝王蟹饗宴五日｜商務艙來回"

**格式結構**:
```
[產品等級][目的地]旅遊｜[賣點1].[賣點2].[賣點3]...[天數]｜[特殊說明]
```

**特徵**:
- 使用「｜」分隔主要區塊
- 使用「.」分隔賣點列表
- 前綴標示產品等級(旗艦/經典/特選)
- 後綴標示特殊服務(商務艙/無購物等)
- 賣點關鍵字密集排列,強調豐富性

### 內容組織架構
1. **頂部資訊區**
   - 產品代碼
   - 可賣席次
   - 出發/回程日期
   - 價格(大號紅色數字)
   - CTA 按鈕(報名)

2. **航班資訊**
   - 參考航班時刻表
   - 航空公司 Logo
   - 去程/回程時間
   - 飛行時長

3. **設計理念/特色說明**
   - 針對高端產品(如商務艙)的詳細介紹
   - 包含香氛、機上娛樂、客艙燈光等細節

4. **住宿資訊**
   - 飯店照片(大圖展示)
   - 飯店特色說明
   - 百選飯店等認證標示

5. **景點介紹**
   - 景點照片(多圖展示)
   - 景點名稱 + 簡短描述
   - 重點景點使用大圖突出

6. **每日行程**
   - DAY 1, DAY 2... 標示
   - 詳細行程描述
   - 餐食安排
   - 住宿資訊

7. **備註說明**
   - 行程備註
   - 房型備註
   - 相關連結

### 圖片排版特點
- **大圖優先**: 主要賣點使用大圖(全寬或半寬)
- **多圖並列**: 景點介紹使用 2-3 張並列小圖
- **圖文結合**: 圖片下方配文字說明
- **比例統一**: 多數圖片使用 16:9 或 4:3 比例

### 資訊呈現方式
- **層級清晰**: 使用標題、副標題、正文三級結構
- **重點突出**: 價格、日期、CTA 使用大號字體和鮮豔顏色
- **標籤輔助**: 使用多種顏色標籤標示產品特性
- **視覺引導**: 使用圖示(Icon)輔助說明
