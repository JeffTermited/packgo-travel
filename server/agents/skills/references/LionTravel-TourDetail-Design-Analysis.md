# 雄獅旅遊行程詳情頁面設計分析

> 完整分析雄獅旅遊行程詳情頁面的設計風格、排版結構、內容組織和互動元素

---

## 一、頁面整體結構

### 1.1 頁面層級架構

```
Header (固定頂部)
├── 會員資訊橫幅
├── Logo + 主導航
└── 麵包屑導航

Hero Section (行程概覽)
├── 行程標題
├── 產品代碼 + 席次資訊
├── 價格 + CTA 按鈕
├── 出發日期日曆
└── 航班資訊

Content Section (主要內容)
├── 搜尋欄 (可再次搜尋)
├── 限時特惠區塊
├── 行程特色 (Highlights)
├── 推薦景點 (Sights)
├── 飯店介紹
├── 餐飲介紹
├── 每日行程
├── 費用說明
├── 注意事項
└── 相關行程推薦

Footer
```

## 二、Hero Section 設計分析

### 2.1 左側:行程概覽卡片

**佈局:**
- 固定寬度卡片 (約 400-500px)
- 白色背景 + 陰影
- 內部元素垂直排列

**元素組成:**
1. **行程標題**
   - 字體大小: 24-28px
   - 字重: Bold
   - 顏色: 深灰色 (#333)
   - 格式: 關鍵字密集堆疊 (用「.」分隔)

2. **產品資訊**
   - 行程編號: 小標籤 (淺藍色背景)
   - 產品代碼: 灰色文字
   - 席次資訊: 綠色文字 (可賣席次)

3. **日期與航班**
   - 去程/回程日期: 大字體顯示
   - 航空公司 logo + 航班時間
   - 飛行時間: 灰色小字

4. **價格與CTA**
   - 價格: 紅色大字 (36px+)
   - 「人/起」小字說明
   - 「費用說明」按鈕 (白色邊框)
   - 「報名」按鈕 (紅色實心,主要CTA)

5. **主圖**
   - 正方形或 4:3 比例
   - 高度約 300-400px
   - 圓角設計 (8-12px)

### 2.2 右側:出發日期日曆

**佈局:**
- 月曆式日期選擇器
- 固定寬度 (約 400-500px)
- 白色背景 + 邊框

**互動設計:**
- 可選日期: 紅色背景 + 價格顯示
- 不可選日期: 灰色
- Hover 效果: 邊框高亮
- 點擊後: 跳轉到報名頁面

## 三、Content Section 設計分析

### 3.1 搜尋欄 (可再次搜尋)

**位置:** Content Section 最上方  
**佈局:** 橫向排列,淺灰色背景 (#F5F5F5)

**元素組成:**
1. **出發地下拉選單** (淺灰色背景)
2. **關鍵字輸入框** (白色背景,佔最大寬度)
3. **日期範圍選擇器** (兩個日期框,中間用 ~ 連接)
4. **搜尋按鈕** (紅色實心)
5. **勾選框** (只找成行、只找可報名)

---

### 3.2 限時特惠區塊

**設計特點:**
- 虛線邊框 (綠色)
- 淺綠色背景
- 圖示 + 標題 + 說明文字
- 使用 ★ 符號強調重點

**內容結構:**
```
★其他注意事項★
※ 優惠價格僅限全程參團...
※ 不同出發日期之售價皆不相同...

★文化深厚 X 三輪車遊古城...
★造型獨特 X COACH飛機航空園區
★竹意盎然 X 漫步於蒼翠秀麗的紫竹谷
...
```

---

### 3.3 行程特色 (Highlights)

**標題設計:**
- 中央對齊
- 圖示 (燈泡) + 「行程特色」文字
- 下方虛線分隔

**內容佈局:**
- 列表式排列
- 每個特色使用 ★ 符號開頭
- X 符號分隔主題和說明
- 範例: `★文化深厚 X 三輪車遊古城，輕鬆探索馬六甲風情。`

---

### 3.4 推薦景點 (Sights)

**標題設計:**
- 中央對齊
- 圖示 (地標) + 「推薦景點」文字
- 下方虛線分隔

**卡片佈局:**
- 網格排列 (3-4 列)
- 每個景點卡片包含:
  1. **景點圖片** (16:9 或 4:3 比例,圓角)
  2. **景點名稱** (Bold,18-20px)
  3. **副標題** (小字,灰色)
  4. **景點描述** (2-3 行,灰色文字)

**卡片樣式:**
- 白色背景
- 陰影效果
- Hover 時: 陰影加深 + 輕微上移

---

### 3.5 飯店介紹

**標題設計:**
- 「保證入住」標籤 (紅色背景)
- 「海上水上屋」副標題
- 飯店名稱 (大字體,Bold)

**內容結構:**
1. **飯店描述** (段落文字,2-3 段)
2. **註記事項** (灰色小字,編號列表)
3. **活動介紹**
   - 免費活動 (列表)
   - 付費活動 (列表)
4. **飯店圖片輪播**
   - 外觀、房型、活動圖片
   - 3-5 張圖片
   - 圓角設計

---

### 3.6 餐飲介紹

**卡片佈局:**
- 網格排列 (2-3 列)
- 每個餐飲卡片包含:
  1. **餐飲圖片** (正方形或 4:3 比例,圓角)
  2. **餐飲名稱** (Bold,18-20px)
  3. **副標題** (小字,灰色)
  4. **餐飲描述** (2-3 行,灰色文字)

**範例:**
```
視覺味覺的雙重享受
古宅娘惹風味

沉浸在古色古香的環境中,感受海峽華人文化中那份對傳統和家的溫暖依戀...
```

**時間軸佈局:**
- 左側: 日期標籤 (圓形,紅色背景)

**行程卡片結構:**
1. **早餐/午餐/晚餐** 標籤 (圖示 + 文字)

  1. **標題** (Bold)

## 四、設計風格總結

### 4.1 色彩系統

**主色:**
- 紅色 (#E60012): CTA 按鈕、價格、重要標籤
- 藍色 (#0066CC): 次要按鈕、連結

**輔助色:**
- 綠色 (#00A651): 成功狀態、可用席次
- 灰色 (#666, #999, #CCC): 次要文字、邊框
- 淺灰色 (#F5F5F5, #FAFAFA): 背景色

**語義色:**
- 紅色: 價格、促銷、CTA
- 綠色: 可用、成功
- 灰色: 不可用、次要資訊

### 4.2 字體系統

**字體家族:**
- 主字體: 微軟正黑體 (Microsoft JhengHei) 或 Noto Sans TC
- 英文字體: Arial, Helvetica, sans-serif

**字體大小:**
- H1 (頁面標題): 28-32px, Bold
- H2 (區塊標題): 24-28px, Bold
- H3 (卡片標題): 18-20px, Bold
- Body (內文): 14-16px, Regular
- Small (次要文字): 12-14px, Regular

### 4.3 間距系統

**Container 寬度:**
- 最大寬度: 1200-1400px
- 左右 Padding: 20-40px

**區塊間距:**
- 大區塊間距: 60-80px
- 中區塊間距: 40-60px
- 小區塊間距: 20-40px

**卡片間距:**
- 卡片間距: 20-30px
- 卡片內 Padding: 20-30px

### 4.4 圓角與陰影

**圓角:**
- 按鈕: 4-8px
- 卡片: 8-12px
- 圖片: 8-12px
- 輸入框: 4-8px

**陰影:**
- 卡片陰影: `0 2px 8px rgba(0,0,0,0.1)`
- Hover 陰影: `0 4px 16px rgba(0,0,0,0.15)`

## 五、互動設計

### 5.1 按鈕狀態

**Primary Button (紅色):**
- Default: 紅色背景 (#E60012)
- Hover: 深紅色 (#CC0000)
- Active: 更深紅色 (#B30000)
- Disabled: 灰色 (#CCC)

**Secondary Button (白色邊框):**
- Default: 白色背景 + 紅色邊框
- Hover: 淺紅色背景
- Active: 紅色背景 + 白色文字

### 5.2 卡片互動

**Hover 效果:**
- 陰影加深
- 輕微上移 (transform: translateY(-4px))
- 過渡動畫 (transition: 0.3s ease)

**點擊效果:**
- 跳轉到對應頁面
- 或展開更多資訊

### 5.3 日曆互動

**日期選擇:**
- 可選日期: 紅色背景 + 價格顯示
- 不可選日期: 灰色 + 不可點擊
- Hover: 邊框高亮
- Selected: 深紅色背景

## 六、響應式設計

### 6.1 桌面版 (>1200px)

- Hero Section: 左右兩欄 (概覽卡片 + 日曆)
- 景點卡片: 3-4 列網格
- 餐飲卡片: 2-3 列網格

### 6.2 平板版 (768px - 1200px)

- Hero Section: 上下兩欄 (概覽卡片在上,日曆在下)
- 景點卡片: 2 列網格
- 餐飲卡片: 2 列網格

### 6.3 手機版 (<768px)

- Hero Section: 單欄垂直排列
- 景點卡片: 1 列網格
- 餐飲卡片: 1 列網格
- 搜尋欄: 垂直排列

---

## 七、關鍵設計原則

### 7.1 資訊層級清晰

1. **最重要**: 價格 + CTA 按鈕 (紅色,大字體)
2. **次重要**: 行程標題 + 日期 (大字體,Bold)
3. **一般資訊**: 景點描述 + 飯店介紹 (中等字體)
4. **次要資訊**: 注意事項 + 費用說明 (小字體,灰色)

### 7.2 視覺引導

- 使用紅色引導用戶注意力到 CTA 按鈕
- 使用圖示和標題分隔不同區塊
- 使用虛線和分隔線增加視覺層次

### 7.3 內容組織

- 從概覽到細節 (Hero → Highlights → Sights → Details)
- 從視覺到文字 (圖片 → 標題 → 描述)
- 從重要到次要 (價格 → 行程 → 注意事項)
