# Sipincollection 設計風格觀察筆記

## 初步視覺分析 (2026-01-26)

### 品牌定位
- **品牌名稱**: 璽品 SP collection (Signature & Presence)
- **品牌理念**: 
  - Signature: 屬於你的獨特旅程風格
  - Presence: 活在當下的感官奢旅
- **目標客群**: 追求體驗感受、注重品質的精緻旅遊客群

### 視覺觀察 (基於首屏)

#### 配色方案
- **主色調**: 深灰色導航列 (#深灰)
- **背景**: 大幅全屏背景圖片 (目的地實景)
- **文字**: 白色標題文字疊加在背景圖上
- **強調色**: 紅色 (用於 Menu 按鈕和重要元素)

#### 排版特點
- **Hero 區域**: 全屏沉浸式背景圖
- **標題層次**: 
  - 主標題: 大字體白色 (北海道二世谷雅奢6日)
  - 副標題: 小字體白色 (行程亮點描述)
- **導航設計**: 固定頂部深灰色導航列
- **按鈕樣式**: 紅色強調按鈕 (Menu)

#### 視覺元素
- **Logo**: 左上角綠色圖案 + 璽品文字
- **圖片使用**: 高品質目的地實景照片
- **文字疊加**: 白色文字直接疊加在背景圖上
- **間距**: 寬敞的留白設計

### 內容區塊設計觀察

#### 特色區塊 (米其林一星鑰旅宿)
- **背景**: 淺灰色 (#F5F5F5 或類似)
- **標題文字**: 
  - 左側: 大型直書中文「雅奢旅宿」(藍紫色 #6B5B95 或類似)
  - 右側: 橫書小標題「米其林一星鑰　尊爵享受」(深灰色)
- **圖片處理**: 
  - 大幅橫向圖片
  - 圖片右側疊加文字說明
  - 文字顏色: 深灰色
- **排版特點**:
  - 左右分欄設計
  - 大量留白
  - 圖文並茂

#### 文字設計
- **中文標題**: 使用直書排版營造東方美學
- **顏色層次**: 藍紫色強調 + 深灰色說明文字
- **字體**: 優雅的襯線字體 (推測為明體或類似字體)
