# 目的地色彩對照表

> **文件版本**: 1.0.0  
> **創建日期**: 2026-01-26  
> **適用範圍**: ColorThemeAgent, ImagePromptAgent  
> **用途**: 根據行程目的地自動選擇合適的主題色彩

---

## 📋 使用說明

本文件提供不同旅遊目的地的建議色彩方案,供 ColorThemeAgent 在生成行程時參考。每個目的地都有主色調、輔助色和強調色的完整色板。

### 使用原則
1. **主色調**: 用於大面積背景或主要視覺元素
2. **輔助色**: 用於次要元素、卡片背景
3. **強調色**: 用於 CTA 按鈕、重要標籤
4. **文字色**: 確保與背景有足夠對比度
