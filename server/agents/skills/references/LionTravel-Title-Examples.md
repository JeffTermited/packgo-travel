# 雄獅旅遊標題範例庫

> 收集自雄獅旅遊官網的真實行程標題,用於 ContentAnalyzerAgent 標題生成參考

## 標題格式特徵

雄獅旅遊的標題採用**關鍵字密集堆疊**格式,特點如下:

1. **促銷標籤前綴**: 寒假過年大促銷~、省最大~、寒假折2千｜
2. **目的地明確**: 高松四國旅遊、日本大阪旅遊、泰國清邁旅遊
3. **景點/體驗堆疊**: 用「.」或「,」分隔,密集列舉 5-10 個景點或體驗
4. **天數後綴**: 五日、六日
5. **無詩意化修飾**: 直述式,不使用比喻或意象

## 標題生成規則 (ContentAnalyzerAgent 應遵循)

### 基本結構
```
[促銷標籤][目的地][景點1].[景點2].[景點3]...[景點N][天數]
```

### 詳細規則

1. **促銷標籤 (可選)**
   - 格式: `寒假過年大促銷~`、`省最大~`、`寒假折2千｜`
   - 使用時機: 有特殊促銷活動時
   - 分隔符: `~` 或 `｜`

2. **目的地 (必要)**
   - 格式: `[國家/地區][城市]旅遊`
   - 範例: `日本大阪旅遊`、`泰國清邁旅遊`、`峇里島旅遊`
   - 可使用 `｜` 補充說明: `日本大阪旅遊│京阪神奈`

3. **景點/體驗堆疊 (必要, 5-10個)**
   - 分隔符: `.` (全形句點)
   - 每個景點: 3-10 個中文字
   - 類型: 景點名稱、體驗活動、飯店等級、美食特色
   - 範例: `奈良東大寺.梅花鹿公園.嵐山渡月橋.北野異人館街.京都錦市場.日本環球影城`

4. **天數 (必要)**
   - 格式: `五日`、`六日`、`七日`
   - 使用中文數字

5. **補充說明 (可選)**
   - 格式: `｜[說明]` 或 `([說明])`
   - 範例: `｜保住三晚溫泉`、`(6人成行/含稅)`

### 禁止使用的元素

❌ **詩意化修飾詞**: 雅奢、秘境、光影、極致、漫遊、走進
❌ **比喻和意象**: 光影之城、藝術家眼中的旅程
❌ **抽象概念**: 禪意、慢活、品味
❌ **形容詞堆疊**: 浪漫的、夢幻的、極致的

### 應使用的元素

✅ **具體景點名稱**: 奈良東大寺、嵐山渡月橋、金刀比羅宮
✅ **明確體驗活動**: 冰釣鱒魚、滑雪趣、採草莓
✅ **飯店/餐食特色**: 升等五星、米其林推薦美食、四大螃蟹
✅ **交通/設施**: 纜車、遊船、海盜船

## ContentAnalyzerAgent 實作指引

### 標題生成流程

1. **提取關鍵資訊**
   - 目的地國家/城市
   - 行程天數
   - 主要景點 (5-10個)
   - 特色體驗
   - 飯店等級
   - 美食特色

2. **組合標題**
   ```typescript
   const title = `${destination}旅遊｜${highlights.slice(0, 8).join('.')}${days}日`;
   ```

3. **驗證標題**
   - 長度: 40-80 個中文字
   - 景點數量: 5-10 個
   - 無詩意化修飾詞
   - 使用 `.` 分隔景點

### 範例程式碼

```typescript
// ❌ 錯誤: 詩意化風格
const badTitle = "北海道二世谷雅奢6日";

// ✅ 正確: 雄獅風格
const goodTitle = "冬季北海道.破冰船.冰釣公魚.丹頂鶴.旭山動物園.熱氣球.四大螃蟹美人湯五日";
```
