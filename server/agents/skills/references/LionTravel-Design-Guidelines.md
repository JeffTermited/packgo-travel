# 雄獅旅遊設計規範 (LionTravel Design Guidelines)

**版本**: 1.0  
**最後更新**: 2026-01-26  
**適用範圍**: PACK&GO 旅行社行程生成系統

---

## 一、品牌核心

### 品牌定位
雄獅旅遊是台灣最大的旅行社品牌,專注於**大眾旅遊市場**,提供高性價比的團體旅遊、自由行、機票訂房等服務。品牌形象強調**專業、可靠、親民**,是台灣家庭旅遊的首選品牌。

### 設計理念
- **清晰易讀**: 資訊層級分明,重點突出,使用者能快速找到所需資訊
- **視覺吸引**: 使用高品質圖片和鮮豔色彩吸引注意力
- **功能優先**: 強大的搜尋和篩選功能,滿足不同需求
- **信任感**: 專業排版和完整資訊建立品牌信賴度

## 二、視覺設計系統

### 配色方案

#### 主色調
```
品牌紅色: #E60012 (雄獅 Logo 紅,僅用於 Logo 和重要 CTA)
深灰色: #333333 (主要文字)
淺灰色: #666666 (次要文字)
背景白色: #FFFFFF
淺灰背景: #F5F5F5 (區塊分隔)
```

#### 輔助色彩
```
橙色標籤: #FF6600 (促銷標籤、經典產品)
藍色按鈕: #0099FF (搜尋按鈕、主要 CTA)
綠色狀態: #00CC66 (成團可報名、無購物)
灰色狀態: #999999 (暫時額滿)
黃色標籤: #FFD700 (限時優惠)
紅色價格: #E60012 (價格數字)
```

#### 色彩使用原則
1. **大量留白**: 背景以白色為主,避免視覺疲勞
2. **品牌紅色節制使用**: 僅用於 Logo、價格和重要 CTA
3. **多色標籤區分**: 使用不同顏色標籤快速傳達產品特性
4. **價格突出**: 使用大號紅色數字突出價格資訊

### 字體系統

#### 字體家族
```
中文字體: Microsoft JhengHei (微軟正黑體) / Noto Sans TC
英文字體: Arial, Helvetica, sans-serif
數字字體: Arial (價格數字)
```

#### 字體大小層級
```
大標題: 24-32px (行程主標題)
中標題: 18-24px (區塊標題)
正文: 14-16px (描述文字)
小字: 12px (標籤、日期、備註)
價格: 24-32px (粗體紅色)
```

#### 字體使用原則
- 標題使用粗體 (font-weight: 700)
- 正文使用常規體 (font-weight: 400)
- 行距: 1.5-1.8 (確保可讀性)
- 避免使用過多字體樣式,保持一致性

### 間距系統
```
容器內距: 16px / 24px / 32px
元素間距: 8px / 12px / 16px
區塊間距: 24px / 32px / 48px
```

## 三、標題風格

### 標題格式結構
```
[產品等級][目的地]旅遊｜[賣點1].[賣點2].[賣點3]...[天數]｜[特殊說明]
```

### 範例
```
✅ 好的範例:
"旗艦北海道旅遊｜保住雙百選溫泉.小樽貴賓館.秘境青池.小樽遊船.騎馬體驗.日本和牛.帝王蟹饗宴五日｜商務艙來回"

"泰國清邁旅遊｜渡假村2晚.金三角遊船.天空步道.白廟.友善象營.泰服體驗.米其林推薦美食.無購物五日"

❌ 不好的範例:
"北海道五日遊" (太簡略,缺少賣點)
"探索北海道的秘境之美,品味和牛與帝王蟹的饗宴" (太詩意,不符合雄獅風格)
```

### 標題撰寫原則
1. **關鍵字密集**: 用「.」分隔多個賣點,展現行程豐富性
2. **前綴標示等級**: 旗艦/經典/特選/smart tour 等
3. **後綴標示特殊服務**: 商務艙/無購物/含稅等
4. **避免詩意化**: 使用直述式,清楚說明行程內容
5. **長度適中**: 建議 40-80 字,避免過長或過短

## 四、圖片使用規範

### 圖片類型
- **行程主圖**: 16:9 橫向照片,展現目的地特色
- **目的地風景**: 自然風光、地標建築、城市景觀
- **活動照片**: 遊客體驗、特色活動、互動場景
- **美食照片**: 當地特色料理、餐廳環境

### 圖片品質要求
```
解析度: 至少 1200x675px (16:9) 或 1200x900px (4:3)
檔案格式: JPEG / WebP
檔案大小: 200KB - 500KB (壓縮後)
色彩飽和度: 中高飽和度,色彩鮮豔
構圖: 主體突出,背景簡潔
```

### 圖片風格特徵
1. **明亮鮮豔**: 色彩飽和度高,吸引眼球
2. **真實感**: 避免過度後製或濾鏡,保持真實性
3. **主體突出**: 構圖清晰,主體佔畫面 40-60%
4. **活潑動感**: 展現旅遊的樂趣和活力

### 圖片排版原則
- **大圖優先**: 主要賣點使用大圖(全寬或半寬)
- **多圖並列**: 景點介紹使用 2-3 張並列小圖
- **圖文結合**: 圖片下方配簡短文字說明
- **比例統一**: 同一區塊的圖片使用相同比例

## 六、內容風格

### 描述文字原則
1. **簡潔明瞭**: 重點突出,避免冗長
2. **關鍵資訊優先**: 天數、出發地、價格、特色
3. **標籤輔助**: 用標籤快速傳達行程特點
4. **避免主觀形容詞**: 少用"美麗"、"夢幻"等詞彙

### 資訊層級
```
1. 行程名稱 (最重要)
2. 價格 (次重要)
3. 出發日期 + 狀態 (重要)
4. 標籤 + 特色 (輔助)
5. 詳細描述 (補充)
```

## 七、AI Agents 應用指南

### ContentAnalyzerAgent
- **標題生成**: 使用關鍵字密集堆疊格式,避免詩意化
- **賣點提取**: 提取 5-8 個核心賣點,用「.」分隔
- **描述撰寫**: 簡潔直述,重點突出

### ColorThemeAgent
- **主色選擇**: 根據目的地選擇 1-2 種主色
- **輔助色**: 使用雄獅標準輔助色(橙藍綠)
- **避免**: 過多色彩使用,保持 2-3 種主色調

### ImagePromptAgent
- **風格關鍵字**: "bright", "vibrant", "colorful", "realistic"
- **構圖要求**: "clear subject", "simple background", "16:9 ratio"
- **避免關鍵字**: "artistic", "moody", "vintage", "film grain"

### ItineraryAgent
- **格式**: DAY 1, DAY 2... 標示
- **內容**: 詳細行程 + 餐食 + 住宿
- **重點**: 景點名稱 + 簡短描述

## 八、品質檢查清單

### 標題檢查

- [ ] 長度是否在 40-80 字?

## 九、與 Sipincollection 的差異

| 項目 | 雄獅旅遊 | Sipincollection |
|------|----------|-----------------|
| 品牌定位 | 大眾市場 | 高端市場 |
| 配色策略 | 多色彩 | 單一主色 + 黑白灰 |
| 標題風格 | 關鍵字密集堆疊 | 詩意化、優雅簡潔 |
| 圖片風格 | 明亮鮮豔、活潑 | 柔和優雅、藝術感 |
| 排版密度 | 資訊密集 | 留白充足 |
| 文案語氣 | 直接、促銷導向 | 優雅、體驗導向 |

**建議**: 當專案需要切換風格時,參考此對比表進行調整。
