## 📋 使用說明

本文件收錄優質的詩意化標題範例,供 ContentAnalyzerAgent 在生成行程標題時參考。避免直白的「XX國N日遊」格式,改用富有美感與想像空間的表達方式。

## 🎨 標題撰寫原則

### 1. 三段式結構
```
[目的地] + [特色形容詞] + [天數]
```

### 2. 優雅詞彙替換表

| 一般用詞 | 詩意替換詞 |
|---------|-----------|
| 豪華 | 雅奢、臻選、華麗、尊榮 |
| 旅遊 | 旅宿、旅程、漫遊、遊歷 |
| 景點 | 秘境、勝地、聖地、仙境 |
| 飯店 | 旅宿、度假村、行館、別墅 |
| 行程 | 旅程、漫遊、探索、巡禮 |
| 美食 | 饗宴、珍饈、佳餚、盛宴 |
| 風景 | 風光、景致、勝景、美景 |
| 文化 | 風情、韻味、底蘊、脈動 |
| 體驗 | 感受、品味、沉浸、領略 |
| 奢華 | 奢逸、華貴、尊貴、雅致 |

## 💡 創意技巧

### 1. 使用感官詞彙
```
視覺: 絕美、壯麗、璀璨、斑斕
聽覺: 悠揚、婉轉、靜謐、寧靜
嗅覺: 芬芳、馥郁、清新、淡雅
味覺: 甘醇、香醇、鮮美、醇厚
觸覺: 柔軟、溫潤、細膩、綿密
```

### 2. 使用文化符號
```
日本: 櫻花、楓葉、富士山、和風、禪意
法國: 薰衣草、艾菲爾鐵塔、浪漫、藝術

希臘: 愛琴海、藍白小屋、神話、古典
```

### 3. 使用季節元素
```
春: 櫻花、新綠、花海、春暖
夏: 碧海、藍天、陽光、熱情
秋: 楓葉、金黃、豐收、涼爽
冬: 雪景、溫泉、極光、靜謐
```
